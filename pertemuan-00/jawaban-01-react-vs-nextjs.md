## 📝 Kuis Singkat

### 1. Next.js adalah...

**Jawaban:** Framework React untuk membangun aplikasi web yang mendukung rendering seperti SSR (Server-Side Rendering) dan SSG (Static Site Generation).

### 2. Mana yang TIDAK perlu di-install manual di Next.js?

**Jawaban:** React dan ReactDOM (karena otomatis terinstall saat membuat project Next.js).

### 3. Saat install Next.js, agar menggunakan JavaScript (bukan TypeScript), kita harus pilih...

**Jawaban:** Pilih No saat ditanya penggunaan TypeScript.

### 4. Apakah komponen React bisa dipakai di Next.js?

**Jawaban:** Ya, bisa. Next.js dibangun di atas React, jadi semua komponen React tetap bisa digunakan.

### 5. Sebutkan minimal 3 fitur yang Next.js berikan di atas React biasa!

**Jawaban:** 1.Server-Side Rendering (SSR)
